When HTML and CSS code are in separate files, the HTML file must know exactly where the CSS code is kept, otherwise, the styling can't be applied the web page. In order to apply the styling to the web page, we'll have to link the HTML file and the CSS file together. You can use the element to link the HTML and CSS files together. The element must be placed within the head of the HTML file. It is a self-closing tag and requires the following three attributes: href - like the anchor element, the value of this attribute must be the address, or path, to the CSS file. type - this attribute describes the type of document that you are linking to (in this case, a CSS file). The value of this attribute should be set to text/css. rel - this attribute describes the relationship between the HTML file and the CSS file. Because you are linking to a stylesheet, the value should be set to stylesheet.
