Over the past few exercises, you've seen CSS examples that use colors like Red, Blue, or Cyan. In CSS, these colors are technically known as named colors. There are a total of 147 named colors. At this point, you might be wondering if you are expected to memorize the list of 147 named colors that CSS offers. Fortunately, you don't have to! There are plenty of available resources on the Internet that list all of the named colors in CSS, like the one we linked you to above.
